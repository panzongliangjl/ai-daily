Anthropic 发布 Cowork：无代码桌面 AI Agent 正式登场

AI Agent 产品发布
事件

Anthropic 本周推出 Cowork，一款可直接在用户本地电脑文件中工作的 Claude 桌面 Agent。无需编程，即可操作文档、表格和本地应用，也可远程监督控制桌面任务。

关键细节
  • 前沿模型的计算机操作能力（导航 UI、点击软件、填写表单）近期在基准测试中已超越人类基线
  • Anthropic 与微软 Copilot、Google Workspace AI 在企业桌面入口展开正面竞争
  • 零门槛形态首次将 Agent 能力触达大众非技术用户
💡 为何关注

这是 AI Agent 从「云端任务执行」向「本地文件操作」延伸的重要节点。任何有界面的软件——包括遗留系统、行业专用工具——都可被 Agent 直接操作，无需 API。Cowork 标志着这类能力以完整产品形态落地。

灵初智能（PsiBot）完成 20 亿元融资，具身智能商业化加速

具身智能 融资动态
事件

成立仅两年的初创公司 灵初智能（PsiBot）完成天使轮 + Pre-A 轮融资，总金额达 20 亿元人民币（约 2.8 亿美元），投资方涵盖国开金融、国中资本、徐汇资本等国家与地方背景资本。

公司技术路线
核心判断

具身智能当前核心瓶颈不是硬件，而是高质量训练数据的规模化收集。PsiBot 押注数据驱动路线，重点构建大规模数据收集系统与可扩展训练管道。

  • 创始人来自京东机器人业务，拥有 20+ 年硬件与商业化经验
  • 「小全栈」模式：自控关键硬件设计，外包生产，平衡控制力与成本
  • 当前落地场景：物流仓储（服装仓库条形码扫描、分拣等半结构化任务）
  • 与北京大学合作建立具身灵巧操作联合实验室
💡 为何关注

此次融资规模与速度印证中国具身智能赛道的资本热度与政策驱动力。「小全栈 + 数据飞轮 + 物流场景优先」的商业模式正成为行业共识模板，后续可观察其能否从试点转化为大规模可盈利部署。

3

Nature 子刊发布 LLM-ROS 开源框架，具身智能通用底座迎新突破

具身智能 学术研究
事件

《自然·机器智能》（3月16日）发表论文，提出将 LLM 与机器人操作系统（ROS）深度集成的通用框架，支持自然语言指令→机器人动作的全链路转化。完整代码已在 Huawei Noah's Ark Lab GitHub（ROSLLM）开源。

核心创新点
  • 双模执行：支持内联代码（inline code）与行为树（behaviour trees）两种执行模式，适应不同复杂度任务
  • 反思机制：机器人能从失败中持续学习，逐步提升任务成功率
  • 全开源 LLM：所有实验均基于开源预训练 LLM 完成，不依赖专有模型，可复现性强
  • 广泛验证：覆盖长周期任务、桌面重排、动态优化、远程监督控制等多场景
开源地址

github.com/huawei-noah/HEBO/tree/master/ROSLLM

💡 为何关注

顶级期刊背书 + 完整开源，意味着这套方案将大大降低学术界和工业界的具身智能开发门槛。模块化、「即插即用」式 LLM 替换设计，让研究者可以快速在不同机器人平台上复用。

来源：Nature Machine Intelligence Huawei Noah's Ark Lab LLM + ROS
4

GTC 2026 收官：NVIDIA Agent 推理架构全面落地

基础设施 AI Agent
事件

NVIDIA GTC 2026（3月16-19日，圣何塞）本周收官。NVIDIA 针对 AI Agent 时代的推理算力体系重构路线完整呈现：Groq 3 LPU + Rubin GPU + Dynamo 软件构成「解耦推理」架构。

关键数字 & 产品
  • Dynamo 软件实现解耦推理，延迟减半，吞吐提升最高 35 倍
  • Groq 3 LPU：SRAM 存储，专攻低延迟解码；Rubin GPU：负责高并发上下文处理
  • NemoClaw 平台：为 Agent 提供安全外壳，集成策略引擎、网络护栏、隐私路由
  • Nemotron 3 Super：Agent 专用模型，对标行业顶尖安全与性能
  • Vera Rubin 平台 + LPX 机架（256 LPU）：支持万亿参数模型推理
背景：为何 Agent 倒逼算力革新

Agent 应用（如 Claude Code）单日 token 消耗比 ChatBot 高一个数量级，旧有以训练为中心的 GPU 架构存在精度冗余、能耗高、延迟高问题，专用推理架构成为必须。

💡 为何关注

NVIDIA 以「解耦推理 + 专用 LPU」重定义 Agent 基础设施的硬件范式，国内推理芯片厂商（寒武纪、海光等）面临重要参照系。Agent 算力需求的爆发，将是下一个算力投资的核心逻辑。

来源：36氪 · 阿尔法公社 NVIDIA GTC 2026 推理算力

Mistral Forge 发布：企业自训前沿模型，破解 Agent 数据隐私瓶颈

企业级 AI Agent
事件

Mistral AI 发布 Mistral Forge 平台，允许企业使用自有私有数据从头训练定制化前沿 AI 模型，直接挑战 OpenAI 和 Anthropic 的企业服务。同日，NVIDIA 发布 Nemotron-3-Nano-4B，一款专为边缘计算设计的 40 亿参数紧凑模型。

关键细节
  • Forge 核心卖点：企业私有数据不上传第三方，完全可控，解决 Agent 落地的数据隐私顾虑
  • 直接挑战 OpenAI Fine-tuning API 与 Anthropic Claude Enterprise
  • Nemotron-3-Nano-4B：4B 参数，适配边缘计算，印证端侧 Agent 成为新布局重点
Agent 落地的关键瓶颈

企业不愿将核心流程数据上传给第三方 API，导致高价值 Agent 场景迟迟无法落地。Forge 代表「自训可控」路线的系统化产品化，是破解这一瓶颈的直接尝试。

💡 为何关注

若 Forge 被市场接受，将加速金融、医疗、法律等高敏感数据行业的企业级 Agent 渗透节奏。同时，Nano-4B 的发布与 Cowork 形成呼应——端侧 + 桌面双线并进，AI Agent 的部署范式正在多元化。
