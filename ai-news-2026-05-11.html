News 01 · AI Agent

Hermes Agent 单日消耗 2710 亿 Token，OpenRouter 榜首易主

在 OpenRouter 全球 Agent 排行榜上，新晋选手 Hermes Agent 凭借「自我学习」功能与激进的生态掠夺策略，单日 Token 消耗量达 2710 亿，成功超越此前霸榜的 OpenClaw，成为平台消耗量第一的 AI Agent。与此同时，Hermes 将小米 MiMo 模型推入其首选模型列表，引发业界对 Agent 生态卡位战格局的高度关注。

  • 规模：单日 2710 亿 Token 消耗，超越 OpenClaw
  • 核心能力：具备「自我学习」机制，可主动优化自身行为策略
  • 生态布局：将小米 MiMo 纳入首选模型，推动跨平台绑定
  • 趋势信号：Agent 赛道生态卡位战已全面开打
🔑 关键数字：2710 亿 Token / 天，相当于约 2040 亿汉字的处理量——Agent 规模化落地已不再是概念。
💡为何关注：Agent 流量规模首次以如此量级出现在公开榜单，意味着 AI Agent 已从「Demo 演示」进入真实大规模使用阶段。谁拿下生态卡位，谁就掌握了下一个平台级入口。

News 02 · AI Agent

OpenAI 发布 GPT-5.5：百万 Token 窗口 + Agents SDK 重磅升级

OpenAI 正式推出 GPT-5.5 系列（含标准版、Pro 版、Instant 版），核心亮点是支持高达 100 万 Token 上下文窗口——是 Gemini 的约 8 倍。同步推出 Agents SDK 重大更新，新增沙箱运行、可检查 Harness 和内存控制三大特性，GPT-5.5 Instant 已作为 ChatGPT 默认模型上线。

  • 上下文窗口：100 万 Token，远超竞品，支持整本书/完整代码库级输入
  • Agents SDK - 沙箱运行：隔离环境构建自主代理，防止恶意代码扩散
  • Agents SDK - 可检查 Harness：开发者可审查 Agent 执行过程，提升可信度
  • Agents SDK - 内存控制：解决旧版内存溢出问题，支持长时间自主任务
  • 赢政指数代码评分：9.5 / 10
🔑 里程碑意义：百万 Token 窗口使 AI 从「短期记忆」跃升为「持久化认知」，Agent 可在不重置上下文的情况下完成跨会话的复杂长任务。
💡为何关注：Agents SDK 的三项升级直接解决了企业级 Agent 从原型走向生产的核心痛点——安全隔离、行为可审计、长任务稳定性。结合百万 Token 窗口，这是 OpenAI 推动 Agent 规模化落地的关键基础设施更新。
来源：winzheng.com OpenAI 2026.05.06

News 03 · AI Agent

OpenAI GPT-Realtime-2 + Google Remy：语音 Agent 与个人代理双向爆发

OpenAI 发布三款实时语音模型：GPT-Realtime-2（GPT-5 级推理能力，实时语音 Agent 场景）、GPT-Realtime-Translate（70 种语言实时翻译为 13 种输出，每分钟仅 ¥0.2）、GPT-Realtime-Whisper（低延迟转录）。与此同时，Google 被曝在员工版 Gemini 内测「24/7 个人代理」Remy，可代替用户完成日程、搜索、邮件等多服务任务。

  • GPT-Realtime-2：听话时边推理边调用工具，处理打断与多步骤任务
  • 语音翻译：70→13 语言实时翻译，成本极低，被称「终结同传」
  • Google Remy：整合 Gmail、Photos、Search、YouTube 历史等 Google 全生态
  • 竞争格局：OpenAI 拥有 OpenClaw 创始人 + 规模；Google 拥有服务深度整合优势
🔑 核心判断：个人 AI 代理的战争已经打响——争夺的不是「谁对话更流畅」，而是「谁能可靠地替用户完成任务并保护数据安全」。
💡为何关注：语音成为 Agent 交互的新入口，而 Google Remy 是第一个可能将「个人助理级 AI 代理」带入数十亿用户日常生活的信号。这意味着 Agent 不再只是开发者工具，而是将深入消费级市场。
来源：36kr IT之家 Business Insider 2026.05.07-08

News 04 · 具身智能

X-Humanoid「玄武 Agent」：业内首个空间记忆 + 人脸识别 + 跨机器人部署平台

具身智能企业 X-Humanoid 发布通用人形机器人智能平台「玄武 Agent（Wise KaiWu）」，经历 14 个月研发，在复杂长程任务中实现 100% 精准率，整体任务完成率 98%+。四大突破：全局场景感知空间记忆、基于 Face ID 的个性化智能、视触觉融合多模态力控、「一脑多机」开放生态架构。

  • 空间记忆：业界首个动态语义地图，视野外物体仍保持持久追踪
  • 个性化：Face ID 识别用户，记忆偏好并主动执行（自动取惯用饮料等）
  • 力控抓取：视觉+触觉融合，跨物体泛化 + 失败自动重试
  • 一脑多机：单一技能包可跨机型部署，技能包体积缩减 80%
  • 应用场景：家庭服务、商业接待、工业操作，均已真实机器人验证
🔑 核心突破：机器人从「被动执行命令」进化为「主动感知 → 持久记忆 → 预判需求 → 物理执行」的完整闭环，真正意义上的「具身智能代理」。
💡为何关注：「一脑多机」架构标志着具身智能从实验室定制化走向批量商业化部署，技能包缩减 80% 大幅降低了厂商接入成本。结合世界模型、VLA、VLM 开源，这是 2026 具身 AI 从量变到质变的标志性节点。

News 05 · 大模型 · 资本

DeepSeek 首轮融资 500 亿人民币，估值 3500 亿，V4.1 定档 6 月

据 The Information 报道，DeepSeek（深度求索）正推进成立以来的首次外部融资，目标金额 500 亿元人民币（约 73.5 亿美元），对应估值约 3500 亿元（21 天内估值翻 5 倍）。创始人兼 CEO 梁文锋个人认购 200 亿元，占本轮 40%。与此同时，DeepSeek V4.1 版本已定档 6 月发布，其战略方向从底层模型供应商转向「全栈基础设施提供商」。

  • 融资规模：500 亿人民币，超越国内四大行上市前战投纪录
  • 创始人领投：梁文锋个人出资 200 亿，占总额 40%
  • 估值跃升：21 天内估值从约 700 亿跃升至 3500 亿，涨幅 5 倍
  • 战略转型：从「开源模型供应商」→「全栈商业闭环基础设施」
  • 下一里程碑：DeepSeek V4.1 计划 2026 年 6 月发布
🔑 资本信号：此轮融资规模创中国 AI 初创企业纪录，意味着全球资本正以「投票」方式确认 DeepSeek 已具备与 OpenAI、Anthropic 正面竞争的实力。
💡为何关注：DeepSeek 从「低价开源」转向「全栈闭环」，是 2026 年大模型格局最重要的战略转折之一。若 V4.1 性能再突破，将对 OpenAI GPT-5.5、Google Gemini 形成直接冲击，重塑全球大模型竞争格局。
